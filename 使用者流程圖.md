# 任務管理應用程式 - 使用者流程圖

## 1. 首次使用流程

```mermaid
flowchart TD
    A[開啟應用程式] --> B[進入任務管理頁面]
    B --> C[看到預設根任務]
    C --> D{選擇操作}
    D -->|建立新任務| E[點擊新增任務]
    D -->|使用模板| F[點擊使用模板]
    D -->|瀏覽現有任務| G[展開任務樹]
    E --> H[填寫任務資訊]
    F --> I[選擇模板]
    G --> J[查看任務詳情]
    H --> K[儲存任務]
    I --> L[從模板建立任務]
    J --> M[編輯或完成任務]
    K --> N[任務建立完成]
    L --> N
    M --> O[繼續使用]
    N --> O
```

## 2. 建立新任務流程

```mermaid
flowchart TD
    A[點擊新增任務] --> B[選擇父任務或根節點]
    B --> C[輸入任務標題]
    C --> D[選擇任務圖示]
    D --> E[填寫任務描述]
    E --> F{需要設定日期?}
    F -->|是| G[設定開始日期和截止日期]
    F -->|否| H[繼續其他設定]
    G --> I{需要提醒?}
    H --> I
    I -->|是| J[新增提前提醒]
    I -->|否| K[繼續其他設定]
    J --> L{需要設定地點?}
    K --> L
    L -->|是| M[選擇或輸入地點]
    L -->|否| N[繼續其他設定]
    M --> O{需要設定重複?}
    N --> O
    O -->|是| P[啟用重複功能]
    O -->|否| Q[儲存任務]
    P --> R[設定重複間隔和單位]
    R --> Q
    Q --> S[任務建立完成]
    S --> T[任務出現在任務樹中]
```

## 3. 使用模板建立專案流程

```mermaid
flowchart TD
    A[點擊使用模板] --> B[開啟模板選擇視窗]
    B --> C[瀏覽可用模板]
    C --> D[選擇模板]
    D --> E[預覽模板結構]
    E --> F{確認使用?}
    F -->|否| C
    F -->|是| G[輸入專案名稱]
    G --> H[輸入專案描述]
    H --> I[從模板建立任務樹]
    I --> J{模板有膠囊標記?}
    J -->|是| K[提示需要填入膠囊]
    J -->|否| L[任務樹建立完成]
    K --> M[進入膠囊設定]
    M --> N[新增文字膠囊]
    N --> O[生成子任務]
    O --> L
    L --> P[專案建立完成]
    P --> Q[開始管理任務]
```

## 4. 管理重複任務流程

```mermaid
flowchart TD
    A[選擇要設定的任務] --> B[進入任務詳情]
    B --> C[找到重複區塊]
    C --> D[啟用重複功能]
    D --> E[選擇重複單位]
    E --> F{選擇單位}
    F -->|天| G[設定每 N 天重複]
    F -->|分鐘| H[設定每 N 分鐘重複]
    G --> I[選擇基準點]
    H --> I
    I --> J{基準點}
    J -->|開始日期| K[以開始日期為基準]
    J -->|截止日期| L[以截止日期為基準]
    K --> M[儲存設定]
    L --> M
    M --> N[任務開始自動重置]
    N --> O[完成任務]
    O --> P[系統記錄到日誌]
    P --> Q[到達重置時間]
    Q --> R[系統自動重置任務]
    R --> S[任務恢復為未完成]
    S --> T[可再次完成任務]
    T --> U{查看日誌?}
    U -->|是| V[展開重複日誌]
    U -->|否| W[繼續使用]
    V --> X[查看日誌列表]
    X --> Y[查看日誌分析]
    Y --> Z[查看完成率趨勢]
    Z --> W
```

## 5. 使用文字膠囊生成任務流程

```mermaid
flowchart TD
    A[選擇要使用膠囊的任務] --> B[進入任務詳情]
    B --> C[找到文字膠囊區塊]
    C --> D[啟用佔位符插入]
    D --> E[新增文字膠囊]
    E --> F[輸入膠囊文字]
    F --> G[儲存膠囊]
    G --> H{需要更多膠囊?}
    H -->|是| E
    H -->|否| I[進入膠囊任務模板設計]
    I --> J{使用預設模板?}
    J -->|是| K[載入預設模板]
    J -->|否| L[設計自訂模板]
    K --> M[編輯任務名稱]
    L --> N[新增根任務]
    N --> O[編輯任務名稱]
    M --> P{需要插入膠囊?}
    O --> P
    P -->|是| Q[點擊 +新增膠囊]
    P -->|否| R[繼續編輯]
    Q --> S[膠囊標籤插入到游標位置]
    S --> T{需要調整位置?}
    T -->|是| U[拖曳膠囊標籤]
    T -->|否| R
    U --> R
    R --> V{需要子任務?}
    V -->|是| W[點擊 + 添加子任務]
    V -->|否| X[完成模板設計]
    W --> O
    X --> Y[選擇生成位置]
    Y --> Z[點擊生成子任務]
    Z --> AA[系統為每個膠囊生成任務]
    AA --> AB[任務出現在任務樹中]
    AB --> AC[開始管理生成的任務]
```

## 9. 管理任務樹結構流程

flowchart TD

## 使用說明

這些流程圖使用 **Mermaid** 語法編寫，可以直接在 GitHub 上使用。

### 在 GitHub 中使用：

1. 建立或編輯 `.md` 文件
2. 複製任何一個流程圖的程式碼（從 `\`\`\`mermaid` 到 `\`\`\``）
3. 貼上到你的 Markdown 文件中
4. GitHub 會自動渲染成流程圖

### 範例：

```markdown
## 首次使用流程

\```mermaid
flowchart TD
    A[開啟應用程式] --> B[進入任務管理頁面]
    ...
\```
```

**注意**：在實際使用時，反斜線 `\` 不需要，這裡只是為了顯示語法。

### 流程圖說明：

- **矩形**：一般操作步驟
- **菱形**：決策點（需要使用者選擇）
- **箭頭**：流程方向
- **文字**：操作描述

這些流程圖從使用者的角度展示了應用程式的各種使用場景，可以幫助使用者理解如何使用這個任務管理系統。
